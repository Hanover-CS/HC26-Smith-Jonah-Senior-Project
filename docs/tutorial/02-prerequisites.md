# 2) Prerequisites

**Knowledge**
- Java 17+
- Spring Boot basics (controllers, configuration, templates)
- Git & GitHub basics
- Optional: Thymeleaf basics

**Installed tools**
- IntelliJ IDEA (or your favorite IDE)
- Java 17+
- Gradle (wrapper provided)
- Git
- A Cloudinary account (free tier is fine)

**Project structure assumptions**
- Your app runs on `http://localhost:8081/` (or 8080 if configured)
- You are using Thymeleaf for server-side views
